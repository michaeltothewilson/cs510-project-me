My Projects

Lyrically

For my Full-Stack Web Development group project, we made a lyric-based music search engine. Find songs by their lyrics. Or even find songs that mention a reference of your choice. You might even find Lyrically helpful for Karoake!

Chatty

For my Internet Networking Protocols class group project, we made a web-chat using socket I/O! Join Chatty with a few friends and give it a try! It works sort of like discord :)

Sleepy

For my Mobile-Health class, I developed a contactless sleep tracker and named it Sleepy. Using an arduino and several sensors, I monitored my sleep patterns as well as environmental data from my bedroom. You can check out the data dashboard I developed using Chart JS. Just please don't just my sleep patterns--it has been an extra rough time to be a student!
